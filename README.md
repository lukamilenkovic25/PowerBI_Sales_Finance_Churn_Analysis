# PowerBI_DataAnalyst_Portfolio
Built 3 end-to-end Power BI dashboards (Sales, Finance, Customer Churn). Performed data cleaning and transformation using Power Query. Created DAX measures for YTD, YoY, MoM, variance, and churn rate. Designed interactive dashboards with slicers, KPIs, and drill-downs. Delivered business insights for sales performance, cost control, and customer retention
